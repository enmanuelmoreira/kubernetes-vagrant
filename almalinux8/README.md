# Clúster de Kubernetes local con Vagrant (AlmaLinux 8)

Aprovisiona un laboratorio de k8s con 1 master y 3 workers.

## ¿Como utilizar?

```sh
git clone https://github.com/enmanuelmoreira/kubernetes-vagrant/almalinux8
cd kubernetes
vagrant up
```

## Requisitos

- CPU 4 núcleos
- RAM 4 GB
- VirtualBox
- Vagrant
